Resell Bot

The Resell Bot was created to use Node.js runtime. As a Walmart employee and a sports card reseller, I was looking for ways to earn side hustle income. This bot would have allowed me to find resellable products sold by Walmart and sell that information to others.

This bot works by taking data from the Ebay and Walmart APIs, matching the UPCs of listings from Ebay with Walmart products, and comparing the prices. If an Ebay listing's price was higher, it would notify users in a Discord server about a possible resellable item.

Throughout this project I gained experience in:

  • JavaScript
  • Node.js
  • Npm packages
  • APIs
  • Web Scraping
